# Brief Summary:
This is a website created for the **Multimodal Assignment** for _ENGL2500H_ at _Iowa State University_. The point of this website is to create a fake game company that is advertising 3 games. The goal is to persuade the user into buying one of the 3 games made by the fake company.

# Tech Stack:
- React
- NextJS
- Tailwind CSS
- Framer Motion
